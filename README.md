# Daniel Castro — Portfolio

Personal portfolio built to showcase SaaS products, full-stack projects, and product-focused engineering work.

Built with **Next.js 15**, **TypeScript**, and **Tailwind CSS v4**, with a focus on performance, clean UI, and scalable structure.

---

## Quick Start

```bash
npm install
npm run dev
```

Runs locally at:

```bash
http://localhost:3000
```

---

## Deployment

Deployed on **Vercel** for fast global delivery and seamless Next.js integration.

To deploy manually:

```bash
npx vercel
```

## Featured Projects

### **Mozes**

SaaS platform for agencies to manage clients, projects, and workflows in one place.

**Stack:** Next.js, Prisma, PostgreSQL, Stripe

* Scalable backend architecture
